0) Проверить установлены ли Node и npm (если нет - установить): \
  Visual Studio > `CTRL + тильда` \
  `node -v` \
  `npm -v` 
 
1) Создать проект: \
  `npx create-react-app chungachanga` 

2) Запустить проект: \
   `npm start`

4) Заменить App.js в проекте на App.js из репозитория \
5) index.js вроде тоже хз \
